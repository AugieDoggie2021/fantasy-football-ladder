# Visual Assets

This directory contains all visual assets for the Fantasy Football Ladder application.

## Structure

- **branding/** - Logos, wordmarks, color palette references
- **icons/** - Navigation icons, status icons, functional icons
- **illustrations/** - Empty states, onboarding illustrations
- **league/** - Tier badges, league banners, team avatar defaults
- **app-icons/** - Favicons, PWA icons, iOS app icons

Refer to `VISUAL_ASSETS.md` in the root for the complete asset checklist.

## Status

Assets will be added as they are created during Phase 8 (Visual Design & Interaction Polish).
