# Visual Assets Checklist

## 4.1 Branding & Logos

### Primary App Logo
- [ ] Full version (text + icon)
- [ ] Variant for light background
- [ ] Variant for dark background
- [ ] Square version for avatars / favicons

### Icon-Only Mark
- [ ] For nav bar, buttons, app icon center
- [ ] Simple enough to be legible at 32×32

### Wordmark / Logotype
- [ ] Horizontal lockup: icon + app name
- [ ] For login screen, landing pages, email headers

### Color Palette Reference
- [ ] Small "brand tile" image (optional) showing main colors and accent

---

## 4.2 App Icons & OS-Level Assets

### Web / PWA / Favicon
- [ ] Favicon 16×16
- [ ] Favicon 32×32
- [ ] Favicon 48×48
- [ ] Web app icon 192×192
- [ ] Web app icon 512×512 (for PWA)

### iOS
- [ ] iOS app icon master 1024×1024 PNG (no rounded corners; Apple handles that)
- [ ] Style: consistent with web logo but iOS-friendly
